What is @iverson/ui?
A zero-dependency vanilla CSS + JS component library for building internal tools with a premium aesthetic.
Does it work with React?
Yes! It works with any framework — React, Vue, Svelte, or plain HTML. Just add the CSS and JS.
Is dark mode supported?
Dark mode is built-in. It auto-detects OS preference or you can toggle with data-theme="dark" on the html element.
